# Service Metrics

Десктоп-дашборд (Angular + Electron) для анализа логов сервиса из CSV.

## Готовый exe

После сборки лежит в `release/`:

- **`ServiceMetrics-portable.exe`** — один файл, ничего ставить не надо. Просто запусти.
- `ServiceMetrics-Setup.exe` — установщик (если собирать через `npm run electron:build`).

### Откуда берутся данные

При запуске приложение ищет `data.csv` **рядом с exe**. Положи свой `data.csv`
в ту же папку, что и `ServiceMetrics-portable.exe`, и нажми ↻ (обновить).
Если файла рядом нет — используется встроенная копия-пример.

Кнопка **«📂 Открыть CSV»** позволяет выбрать любой другой файл вручную.

### Какие колонки понимает

Парсер ориентируется на исходные имена логов и подхватывает то, что есть:
`id_service_log, cd_service, dt_tm_event, level, logger, message, ip_requester,
exception, url, nr_dic, cd_dic_status, cd_http_status`.
Отсутствующие колонки (напр. `stack_trace`, `logger`, `exception`) просто игнорируются —
графики и фильтры подстраиваются под реальный набор данных.

## Что внутри

- **KPI**: всего запросов, ошибки и их доля, доля успешных 2xx, уникальные IP/эндпоинты, период.
- **Графики**: динамика запросов и ошибок во времени (час/день/месяц), уровни логов,
  классы и коды HTTP-статусов, топ эндпоинтов (`nr_dic`), топ источников (IP), топ URL, по сервисам.
- **Фильтры**: диапазон дат + быстрые периоды, поиск по тексту, IP, сервис, уровень,
  класс HTTP, эндпоинт, «только ошибки».
- **Таблица**: сортировка по столбцам, пагинация, разворот строки с деталями, экспорт отфильтрованного в CSV.
- **Тема**: переключатель светлая/тёмная (запоминается).

## Разработка

```bash
npm install
npm run electron:dev      # ng serve + Electron с горячей перезагрузкой
npm run start             # только Angular в браузере (http://localhost:4200)
```

## Сборка

```bash
npm run electron:portable # только portable .exe  -> release/
npm run electron:build    # portable + установщик NSIS
```

> Electron закреплён на версии **37.2.0** (бинарь уже в локальном кэше).
> Обновление до более новой версии потребует докачать ~120 МБ бинарника Electron.
